# Minesweeper TUI written in C
- A terminal based minesweeper clone in C

## Usage
1. Clone the repo
2. Build and run
```console
$ gcc ./minesweeper.c -o minesweeper
$ ./minesweeper
```

### Controls
| Key              | Action        |
| ---------------- | ------------- |
| <kbd>W</kbd>     | Move Up       |
| <kbd>A</kbd>     | Move Left     |
| <kbd>S</kbd>     | Move Down     |
| <kbd>D</kbd>     | Move Right    |
| <kbd>F</kbd>     | Flag a cell   |
| <kbd>space</kbd> | Reveal a cell |
